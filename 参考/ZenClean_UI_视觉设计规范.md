# ZenClean 视觉设计与 UI 组件规范 (v0.2.0)

本规范整理自 ZenClean 项目，旨在沉淀一套**赛博机甲 (Cyber-Mecha) 风格**的 Flet UI 设计体系。

---

## 🎨 核心色彩体系 (Color Palette)

采用了深色背景下的三原色变体，强调高对比度与荧光发光感。

| 变量名 | 色值 (Hex) | 视觉用途 | 设计语义 |
| :--- | :--- | :--- | :--- |
| `COLOR_ZEN_BG` | `#0F1115` | 背景层 | 极速冷空背景，深邃而不完全黑 |
| `COLOR_ZEN_SURFACE` | `#171A21` | 卡片层 | 区块容器色，与背景拉开层次 |
| `COLOR_ZEN_PRIMARY` | `#009688` | 核心装饰 | 品牌主色，采用玉石绿，沉稳且专业 |
| `COLOR_ZEN_GOLD` | `#D4AF37` | 尊贵/品牌 | VIP 皇冠色、檀木金，用于高价值反馈 |
| `COLOR_ZEN_DANGER` | `#E74C3C` | 警告/错误 | 朱砂红，用于删除、异常或严重风险 |
| `COLOR_ZEN_TEXT_MAIN` | `#E6EAF0` | 主文字 | 星尘白，主标题，保证极高可读性 |
| `COLOR_ZEN_TEXT_DIM` | `#8B93A6` | 副文字 | 冷峻灰，用于说明文本、禁用态 |

## 🧩 标志性 UI 组件模式

### 1. 「左右护法」HUD 翼板 (Symmetric Capsule)
**场景**：仪表盘左右对称的数值监控。
*   **结构**：Container + Column ([Icon], [Consolas Value], [Label])
*   **交互**：带透明背景色 (`with_opacity(0.05)`) 和单侧加重边框 (`border.only`)。
*   **几何**：非对称圆角设计（如一侧 15px，另一侧 5px）营造机甲部件的拼接感。

### 2. 「四大金刚」功能磁贴 (Function Tiles)
**场景**：核心功能入口 2x2 网格。
*   **特性**：
    *   `ink=True`：强制实装 Material 水波纹点击反馈。
    *   `on_hover` 缩放：在 `on_hover` 中通过变换容器背景色或阴影制造动态感。
    *   **极限间距**：为了增加科技紧凑感，Padding 建议在 10-15px 之间。

### 3. 「机甲心脏」扫描核心 (Core Button)
**场景**：主操作入口。
*   **特性**：
    *   `gradient`：采用 `#1DD1A1` 到 `#00C2FF` 的线性渐变。
    *   `shadow`：通过 `BoxShadow` 制造大范围的青色溢出光晕（发光感）。
    *   `ink=True`：保证圆形水波纹充满整个按钮。

### 4. 「HUD 数据甜甜圈」 (Glow Donut)
**场景**：主视觉容量展示。
*   **特性**：
    *   **分层堆叠**：底层使用 `Container` 制造背景阴影光晕，中层放置 `PieChart`，顶层覆盖 `Column` 用于数值展示。
    *   **等宽字体**：核心数值必用 `Consolas`，字号 `36` 以上。

### 5. 「极速雷达」流式列表 (Stream List)
**场景**：实时动态排行榜。
*   **特性**：
    *   **描边风格**：背景透明，仅保留 `#2C3440` 的深色边框。
    *   **流式加载**：配合后端 Generator 实现“边扫边跳”的排行榜动效。
    *   **图标点缀**：左侧固定图标 + 右侧操作图标组。

## 📏 布局与排版原则

1.  **众星拱月**：核心操作（如扫描按钮）永远居中，四周环绕对称的监控数据。
2.  **字体分工**：
    *   **数值**：统一使用 `Consolas` 或等宽字体，增加“精密仪器”既视感。
    *   **文本**：中文使用系统默认屏显黑体，保持简洁。
3.  **负空间管理**：在深色背景下，使用 1 像素的 `COLOR_ZEN_DIVIDER` (#11FFFFFF) 进行分割，而非粗重的线条。

## 🚀 开发者快速复用逻辑 (Flet Python)

```python
# 典型机甲风容器模板
ft.Container(
    content=ft.Column([...]),
    bgcolor="#15181E",
    border_radius=16,
    border=ft.border.all(1, "#2C3440"),
    padding=15
)
```
