# 更新日志 (Changelog)



## [0.1.4-beta] - 2026-03-11
### 重构
- 系统文件夹搬家引擎从"直接写注册表+Junction"重构为 Windows 原生 SHSetKnownFolderPath API。
- 等同于用户手动通过"属性 → 位置 → 移动"操作，彻底消除路径缓存不刷新的问题。

### 安全加固
- 应用级搬家 (app_migrator.py)：Junction 创建失败时自动将数据回滚至 C 盘原位置。
- 系统级搬家 (system_migrator.py)：Junction 创建失败时自动将数据回滚至 C 盘原位置。
- 系统级还原：文件回迁失败时自动重建 Junction 以保持系统可用性。
- 断点恢复路径：Junction 失败也同样具备自动回滚能力。

## [0.1.0-beta] - 2026-03-02

### 新增

- 增加了基于 PyInstaller 的原生 Splash 启动屏。

- 增加了 `start.bat` 启动辅助脚本。

- 增加了 VC++ 运行库缺失的智能拦截与手动跳转引导。

- 增加了 `src/config/version.py` 用于全局版本号管理。



### 优化

- **启动性能**: 异步化 NTP 校验流程，UI 渲染显著提速。

- **视觉体验**: 窗口初始化顺序重构，彻底消除白屏闪烁。

- **鲁棒性**: 修复了 `os` 模块引用和 `self.page` 挂载时机导致的崩溃。

- **提权逻辑**: 完善了 UAC 失败时的错误记录与降级提示。



### 修复

- 修复了点击 VIP 图标可能导致界面无响应的逻辑错误。

- 修复了图片资源在打包后的 `_internal` 目录下路径解析不一致的问题。
