# ZenClean 用户安装与运行指南

感谢您使用 **禅清 (ZenClean)** —— 基于 AI 的 Windows C 盘极速清理大师。

本文档将指导您如何在新环境（尤其是从未安装过开发环境的普通 Windows 电脑）中成功运行本程序。

## 1. 系统要求
* **操作系统**：Windows 10 / Windows 11 (64位)
* **磁盘空间**：至少 150MB 可用空间（用于存放解压后的程序主体及运行日志）。
* **网络环境**：建议保持联网（用于激活校验与获取最新的 AI 云端清扫规则）。

## 📦 测试版 (Beta) 运行流程

如果您获得了 `ZenClean_0.1.0_Beta.zip` 发布包，请按以下步骤操作：

1. **解压**：将 ZIP 文件夹放入非系统核心目录（如 `D:\Tools`）。
2. **运行**：双击文件夹根目录下的 `start.bat`。
3. **授权**：如果是首次运行且拥有激活码，请在左侧“VIP 激活”处输入开启离线保护。
4. **依赖**：若提示“缺少运行库”，点击“是”跳转至微软官网下载。

## 🛠️ 源代码运行 (开发者)

### 🛡️ 为什么需要管理员权限 (UAC)?
ZenClean 旨在进行**系统级**的深度扫描（例如 `C:\Windows\Temp` 等隐藏锁死区）。
为了保证清理引擎的威力和数据完整性，**程序必须以 Administrator (系统管理员) 身份运行**。

* **运行表现**：当您双击 `ZenClean.exe` 时，屏幕可能会变暗，并弹出一个带有黄色盾牌图标的“用户账户控制 (UAC)”询问窗口：*“你要允许此应用对你的设备进行更改吗？”*
* **您的操作**：请务必点击 **“是”**。程序将会被系统赋予最高权限，随后会自动加载主界面。

## 3. 常见启动报错与解决方案

### ❌ 报错："找不到 MSVCP140.dll" 或 "VC++ 组件缺失"
* **原因**：我们的 UI 渲染引擎 (Flet) 依赖于微软的 C++ 基础运行库。非常纯净且从未玩过大型游戏的 Windows 新系统可能缺漏此组件。
* **解决办法**：请前往微软官方下载并安装 [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) (64位版本)。安装完成后无需重启，再次运行 `ZenClean.exe` 即可。

### ❌ 界面只显示一个无响应的黑窗口或持续闪退
* **建议**：这可能与您电脑的显卡驱动（OpenGL渲染支持）有关。请检查您的显卡驱动是否已经正常安装。
* **应急排查**：您可以尝试进入 `C:\Users\您的用户名\AppData\Roaming\ZenClean\logs\` 目录，查看最新的 `.log` 文件寻找具体的崩溃原因，或将其截图发给技术支持。
