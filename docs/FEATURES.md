## 一、产品定位

ZenClean（禅清）是一款面向 Windows 系统的深度清理与极致优化工具。与市面上常见的"清理神器"不同，ZenClean 聚焦于**安全性、透明度与底层效能**三大核心优先级——不虚报垃圾量，不碰您的登录态和表单历史，不做任何隐蔽的后台联网行为。

### 产品特点

- **AI驱动**: 结合本地规则引擎与云端AI大模型，智能识别文件风险等级
- **安全至上**: 多重保护机制，包括白名单防护、隔离沙箱、操作确认等
- **零感知迁移**: 利用 NTFS Junction 技术，将应用数据透明迁移至其他盘符
- **现代化界面**: 基于 Flutter 的流畅、美观的用户界面

### 2.1 🤖 智能扫描引擎 (Scanner 2.0)

#### 功能描述

ZenClean 的心脏。基于彻底异步隔离架构构建的双核扫描引擎，帮助用户安全地清理C盘垃圾文件。

#### 关键技术特性

| 特性 | 说明 |
|------|------|
| **UI 零阻塞** | 扫描任务跑在独立的操作系统线程中，通过内部消息队列异步向 UI 投递进度信号，界面操作始终流畅 |
| **本地知识库** | 内置超过 40 条基于正则表达式的文件路径匹配规则（存储于 `file_kb.json`），涵盖系统临时区、浏览器缓存、WER 错误报告等高频清理场景 |
| **AI 云端研判** | 对本地规则库未能命中的模糊路径，系统可选择性地调用配置好的大语言模型 API 进行二次智能评估，给出风险评分与清理建议 |
| **风险分级标注** | 每一条扫描结果均经过多级风险标注，系统关键文件有视觉高亮保护 |
| **AI 缓存复用** | 每条路径的 AI 研判结果会被持久化至本地 JSON 缓存，避免对同类文件重复联网消耗 API 配额 |

#### 风险等级分类

| 风险等级 | 标识 | 说明 | 默认操作 |
|---------|------|------|---------|
| LOW | 🟢 低风险 | 安全删除，如临时文件、缓存等 | 可直接清理 |
| MEDIUM | 🟡 中风险 | 建议隔离，需谨慎处理 | 先隔离后清理 |
| HIGH | 🟠 高风险 | 需用户确认，可能影响应用功能 | 用户确认后操作 |
| CRISIS | 🔴 危险 | 系统关键文件 | 永不删除，受白名单保护 |
| UNKNOWN | ⚪ 未知 | 无法判断 | 需用户判断 |

### 2.2 🛡️ 时光机双核隔离舱（Quarantine Sandbox）

#### 功能描述

隔离沙箱是 ZenClean 的安全网功能，提供 72 小时"反悔期"的软删除机制。所有被 ZenClean 清理的文件并不会立即物理销毁，而是被安全地移送至一个隐藏的专属沙箱目录中，给用户充足的余地来恢复误删文件。

#### 操作流程

```
清理操作触发
    ↓
文件移入隔离目录（隐藏属性）
    ↓
元数据写入隔离注册表（quarantine_registry.json）
    ↓
72 小时内 → 支持一键原路恢复
    ↓
超出时限 → 后台守护线程自动静默粉碎
```

#### 核心特性

**智能磁盘选择**
- 自动扫描多个分区
- 选择非系统盘中剩余空间最大的磁盘作为沙箱载体
- 防止 C 盘被二次占据

**72小时后悔期**
- 被清理的文件在隔离区保留72小时
- 用户可随时恢复误删文件
- 过期文件自动清理

**完整的文件追踪**
- 使用注册表记录隔离文件元信息
- 保存原始路径、隔离时间、文件大小
- 支持批量恢复/删除

**隔离区位置**
**隔离区位置**

沙箱自动创建于剩余空间最大的非系统盘根目录：
如 `D:\.ZenClean_Sandbox\`（具体位置由系统根据磁盘容量自动动态决定）

#### 主要功能

- **一键全量恢复**：支持将当前沙箱内的所有文件批量原路恢复
- **一键彻底清空**：手动触发提前销毁，立即释放沙箱占用的空间
- **单项精细管理**：可对每个隔离项目独立执行"恢复"或"彻底删除"操作
- **自动过期清理**：后台守护线程（延迟 15 秒启动）定期巡检并自动销毁超期文件

#### 使用场景

1. **误删恢复**: 清理后发现问题，可一键恢复
2. **文件预览**: 在恢复前确认文件内容
3. **空间管理**: 查看隔离区占用空间，选择性清理

### 2.3 🚚 大厂应用无损极客搬家（App Migration）

#### 功能描述

专门解决"大型应用（微信/QQ/Docker/Chrome 等）占用 C 盘"的痛点。利用 Windows NTFS Junction（目录联接）技术，将应用数据目录从 C 盘物理迁移至其他磁盘，同时在原路径保留透明软链接，**应用层完全无感知，无需重装、无需手动修改任何设置**。

#### 技术原理

```
原路径: C:\Users\xxx\Documents\WeChat Files
目标路径: D:\ZenClean\Migration\WeChat Files
          ↓
创建 Junction: C:\Users\xxx\Documents\WeChat Files → D:\...\WeChat Files
          ↓
应用访问原路径 → 系统自动重定向到新位置 → 应用无感知
```

#### 支持迁移的应用

| 应用 | 迁移内容 | 典型节省空间 |
|------|---------|-------------|
| 微信 | 聊天文件、图片、视频 | 5-50GB |
| QQ | 聊天记录、接收文件 | 3-30GB |
| Docker Desktop | 镜像、容器数据 | 10-100GB |
| VSCode | 扩展插件目录 | 500MB-5GB |
| Google Chrome | 缓存目录（支持细粒度子目录迁移，不影响登录态） | 1-10GB |
| Microsoft Edge | 缓存目录 | 1-10GB |

#### 迁移流程（5阶段原子性状态机）

```
[预检查] → [文件复制] → [完整性校验] → [创建Junction] → [完成]
   ↓           ↓              ↓              ↓
 检查空间    复制文件       MD5校验        建立链接
 检查权限    显示进度       确保完整       删除原文件
 检查冲突
```

每个阶段结果都被持久化到磁盘，断电或强杀后重启应用即可感知中断状态，支持"继续迁移"或"一键无损回滚"。

#### 安全防护机制

- **Pre-flight 容量预检**：搬家前自动校验目标磁盘剩余容量，不足则立即中止
- **进程探测与优雅挂起**：检测到相关进程在运行时，尝试引导用户关闭应用，再执行底层操作，防止文件锁冲突

#### 增量监控

已完成搬家的目录，应用程序会开启增量体积监控。当增长量超过 1 GB 时，看板页面对应卡片将呈现**琥珀色警告**，用户可直接在卡片上点击"一键再迁"，将新增数据同步搬运至目标盘。

#### 断点续传

- 支持检测中断的迁移任务
- 可选择继续迁移或回滚
- 确保数据安全不丢失

### 2.4 ⚡ $PatchCache 与 Windows 更新深度清理（Patch Analyzer）

#### 功能描述

针对 Windows 长期累积的、正常清理工具无法触及的"隐藏重型垃圾"进行深度分析与清理，可释放数 GB 甚至数十 GB 的空间。

#### 清理对象

**Windows Installer $PatchCache$**
- 位置: `C:\Windows\Installer\$PatchCache$`
- 作用: Windows 更新补丁缓存
- 典型大小: 5-20GB
- 风险等级: 高
- 深入目录对安装时间超过 365 天的陈旧补丁进行风险鉴定
- 通过托管 `msiserver`（Windows Installer）服务进行底层控制

**DISM 组件存储库清理**
- 使用 DISM 命令清理更新备份
- 安全移除旧版本组件
- 典型节省空间: 3-15GB
- 命令: `dism /online /cleanup-image /StartComponentCleanup`

**Delivery Optimization 缓存**
- 清理 Windows 更新 P2P 传输功能遗留的已下载分片

#### 操作流程

1. 停止 Windows Installer 服务 (msiserver)
2. 执行文件复制和 Junction 创建
3. 重启服务
4. 验证完整性

#### 安全警告

> ⚠️ **风险提示**：此模块涉及对 Windows 系统服务进行底层控制操作，建议由有经验的用户在执行系统备份后使用。

### 2.6 🔌 系统底层深度整合

#### 2.6.1 Windows 右键菜单集成

- 于注册表 `HKCU` 层级注册右键菜单项，**无需管理员权限**
- 支持三种场景挂载：文件夹右键、文件夹背景右键、磁盘根目录右键
- 右键一键触发扫描，自动将所选路径传入并调起 ZenClean 主窗口

#### 2.6.2 系统盘爆仓气泡预警

- 通过 Windows 任务计划程序 (`Task Scheduler`) 注册定时任务，定期静默探查 C 盘使用率
- 当占用率超过用户设置的预警阈值（默认 90%），推送**原生 Windows Toast 通知**，提醒用户及时清理
- 预警阈值可在"系统控制台"页面通过滑块精细调整（50%~99%）
- 零后台进程常驻，完全依托系统原生任务调度机制

#### 2.6.3 系统级开机自启守护

- 通过 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` 通道写入注册表实现**开机自启**，无需管理员权限
- 程序启动后最小化驻留**系统托盘 (Tray)**，提供右键上下文菜单：可快速扫描当前选中目录、显示主窗口、退出应用
- **拦截关闭操作**：点击窗口关闭按钮时，窗口隐藏而非真正退出，保持后台守护能力

## 三、安全保护机制

### 3.1 白名单防护

系统关键路径被永久保护，永不扫描或删除：

**受保护的系统目录**
- `C:\Windows\System32`
- `C:\Windows\SysWOW64`
- `C:\Windows\WinSxS`
- Windows Defender 目录
- EFI/Boot 分区

**受保护的文件类型**
- `*.sys` - 系统驱动
- `*.dll` - 动态链接库
- `*.exe` - 可执行文件
- `*.msi` - 安装包
- `bootmgr` - 引导管理器
- `pagefile.sys` - 页面文件
- `hiberfil.sys` - 休眠文件

### 3.2 License 激活体系

- 支持在线激活（通过商业授权网关 `hw-license-center` 验证卡密）
- 激活信息通过 JWT (HS256 签名) 加密后持久化到本地 `.dat` 文件
- 后台授权巡检线程（每 30 分钟一次）静默复核云端授权状态，检测到 `[REVOKED]` 标记时自动降级

### 3.3 API 接口防护

**HMAC-SHA256 签名**
- 所有云端请求都带有签名
- 包含时间戳和随机数防重放攻击

**JWT认证**
- VIP功能需要有效的JWT令牌
- 令牌绑定设备硬件ID
- 防止令牌盗用

### 3.4 客户端限流

- 限制: 60秒内最多10次AI请求
- 防止API滥用
- 优雅降级到本地规则

### 3.5 PowerShell 注入防御

- 所有动态拼接进入 `subprocess` 或 `PowerShell` 命令行的用户路径，均经过 XML/HTML 特殊字符转义
- 防止恶意 payload 注入

## 九、 竞品对比分析

### 9.1 系统清理类

| 软件 | 特点 | 与 ZenClean 对比 |
| :--- | :--- | :--- |
| **CCleaner** | 老牌清理工具，知名度高，支持注册表清理 | 商业化严重，曾有隐私争议；无 AI 分析，无应用迁移 |
| **BleachBit** | 开源免费，支持多平台，清理彻底 | 界面较简陋，无中文优化，无隔离沙箱 |
| **Wise Disk Cleaner** | 轻量级，有免费版，支持计划任务 | 无 AI 研判，无应用搬家功能 |
| **Dism++** | 国产开源，系统级清理强，无广告 | 界面偏技术向，无隔离沙箱，无云端 AI |
| **Glary Utilities** | 功能全面，含一键优化 | 商业软件，无应用迁移 |
| **华为电脑管家** | 华为电脑内置，有 C 盘清理 | 仅限华为设备，功能较基础 |

### 9.2 应用迁移类

| 软件 | 特点 | 与 ZenClean 对比 |
| :--- | :--- | :--- |
| **Steam Mover** | 早期 Junction 迁移工具，专为 Steam 游戏设计 | 功能单一，仅支持手动操作，无 UI 引导 |
| **Link Shell Extension** | 资源管理器右键扩展，手动创建 Junction | 需手动操作，无应用感知，适合技术用户 |
| **Folder Move** | 简单的文件夹迁移工具 | 无应用预设，无断点续传，无增长监控 |
| **腾讯电脑管家-软件搬家** | 腾讯出品，支持 QQ/微信迁移 | 有广告，仅支持腾讯应用，无隔离沙箱 |
| **360 安全卫士-C 盘搬家** | 360 出品，知名度高 | 广告多，捆绑严重，隐私争议 |
| **FreeMove** | 开源 Junction 工具，界面简洁 | 无中文，无应用预设，无进度追踪 |

### 9.3 AI + 系统工具类

| 软件 | 特点 | 与 ZenClean 对比 |
| :--- | :--- | :--- |
| **Microsoft PC Manager** | 微软官方，集成 Windows，无广告 | 功能较保守，无深度清理，无应用迁移 |
| **Lenovo Vantage** | 联想官方，硬件+系统优化 | 仅限联想设备，功能受限 |

## 十、 ZenClean 的差异化优势

| 特性 | ZenClean | 竞品现状 |
| :--- | :--- | :--- |
| **AI 智能研判** | ✅ 本地规则 + 云端 LLM 双重分析 | 大多数竞品仅基于规则匹配 |
| **隔离沙箱** | ✅ 72 小时后悔期，智能磁盘选择 | 罕见，多数直接删除 |
| **零感知迁移** | ✅ Junction 技术，应用完全无感 | 部分有，但无断电续传和增长监控 |
| **国产化适配** | ✅ 针对微信/QQ/钉钉/飞书深度优化 | 多数国外软件无此适配 |
| **无广告纯净** | ✅ 开源 MIT 协议 | 部分商业软件广告较多或有隐私争议 |
| **白名单防护** | ✅ 系统关键文件永久保护 | 多数有基础保护，但不完善 |
| **磁盘爆仓预警** | ✅ 原生 Toast 通知，零后台驻留 | 罕见 |
| **右键菜单集成** | ✅ 无需管理员权限 | 部分有 |
