# ZenClean (禅清)

现代 Windows C 盘深度清理与优化工具。

## 特性
- 🧼 **深度清理系统垃圾**
- 📦 **自动清理 npm/pip 等开发缓存**
- 🚀 **智能迁移大文件 (软连接)**
- ✨ **基于 Flet 的极简现代 UI**
- 🧠 **端云协同的 AI 智能判定引擎**

## 文档指引
- 📖 [AI 版图测绘算力（额度）消耗与计算机制](docs/AI_QUOTA_MECHANISM.md)

## 快速开始

1. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```
2. 运行程序:
   ```bash
   python src/main.py
   ```
