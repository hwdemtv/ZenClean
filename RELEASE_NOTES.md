# ZenClean v0.1.0 Beta 发布说明

## 🎉 发布日期
2026年03月02日

## 🚀 极速上手
1. 下载并解压 `ZenClean_0.1.0_Beta.zip`。
2. **双击运行 `ZenClean.exe`**。
3. 若提示“缺少运行库”，按引导完成 Visual C++ 安装。

## ✨ 核心亮点
- 🧼 **AI 深度清理**: 智能识别系统冗余，不再误删。
- ⚡ **秒级启动**: 引入原生 Splash 闪屏，告别启动真空期。
- 🛡️ **安全第一**: 内置多重白名单与 UAC 提权逻辑，保护核心系统文件。
- 🌐 **离线支持**: 联网激活一次，内网长期可用。

## 🛠️ 技术改进
- 统一版本管理机制 (`version.py`)。
- 资源路径动态解析，支持任意目录解压运行。
- 增强的离线 NTP 时间偏差容错处理。
- 自动清理后台扫描子进程，不占系统资源。
- 鉴权配置优化：在 `settings.py` 中内置正式授权服务默认地址，并通过 `python-dotenv` + PyInstaller
  将根目录 `.env` 一同打包到发行版中，确保用户无需手动配置环境变量即可正常联网激活。

## ⚠️ 注意事项
- 本版本为 **Beta 内测版**，主要用于收集清理规则反馈。
- AI 增强分析功能在离线状态下将降级为本地规则模式。
